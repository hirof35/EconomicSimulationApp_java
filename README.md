📘 EconomicSimulationApp_java — 経済シミュレーションアプリ
🎯 目的（Purpose）
このアプリは 経済シミュレーションを目的として開発されました。
ユーザーが 経済の因果構造・政策効果・成長モデルの理解を深められるよう設計されています。

🧩 構造（Structure）
アプリは以下の構造で動作します：

入力（Input）

スライダーによる政策パラメータ操作

AI 技術進歩率

教育・AI 投資率

定年延長効果 など

処理（Logic）

モンテカルロ法による経済成長シミュレーション

ランダム性を含む複数パターンの経済推移

パラメータ変化による成長率の比較

出力（Output）

JavaFX グラフ描画

結果の画像保存

シミュレーションログ（必要に応じて）

状態管理（State）

現在のパラメータ値

グラフの状態

シミュレーション履歴

UI / UX

スライダー中心の直感的な操作

結果が即座に反映されるリアルタイム UI

学習を妨げないミニマルデザイン

構造は 普遍的な経済理解を誰でも得られるよう最小化されています。

🛠 使用技術（Tech Stack）
言語：Java

フレームワーク：Oracle Java

UI ライブラリ：JavaFX

開発思想：構造の明確化・因果の可視化・教育向け UI

📦 機能（Features）
AI 技術進歩率の操作  
技術成長が経済に与える影響を可視化

AI / 教育への投資率操作  
投資が長期成長に与える効果をシミュレーション

定年延長効果の操作  
労働人口の変化が経済に与える影響を表示

画像保存機能  
グラフを PNG 形式で保存可能

グラフリセット機能  
シミュレーション結果をクリアして再実行できる

必要に応じて機能追加が可能な 拡張性の高い構造になっています。

🚧 例外・エラー処理（Exceptions）
アプリでは以下の例外を局所化して処理しています：

入力が空の場合

ファイル保存時にファイルが存在しない / 書き込めない場合

ネットワーク関連の例外（必要な場合）

不正なパラメータ値の入力
